Creating Brackets

Create a new bracket by clicking the Create Bracket button. This will use the default (recommended) settings. If you want to customize how the bracket is generated, use the Options button.

Creating with Options

When creating brackets using the Options button your choices are:

  • Madness
  • Winner
  • Second
  • Algorithm

Madness

Increasing the madness level of the tournament increases the entropy of the tournament. When a higher seed plays a lower seed, the chance the higher seed will upset the lower seed is is increated as the madness increates. The default value of one generates the average amount of madness seen over the last 20 years (an average of 18 upsets per tournament). As the madness number is adjusted, the number of upsets will go up or down accordingly.

Winner

Force the selected team to win the tournament. The software will run tournament simulations until the selected team is found to be the winner.

Second

Force the selected team to be the runner-up. The software will run tournament simulation until the selected team is found to be the runner-up.

Algorithm

Use the selected algorithm to determine winners during tournament simulation. The default Seed Odds is the only viable choice at the moment. The alternative Fifty Fifty gives each team a 50% chance of winning any game, and is only used to demonstrate a pluggable algorithm interface. {% endblock %}
