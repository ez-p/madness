Choosing the Year

By default the web application uses the current year. If you would like to create brackets for a previous year you can select a different year on the Options page. At the bottom of the Options next to the text "Switch years", choose a different year.

Cookies

If you allow cookies, once you select a specific year that year will remain selected until a different year is selected.
