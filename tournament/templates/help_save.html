Saving Brackets

Login

First, ensure you have created a user account and are logged in. If the Keep button is disabled, you are not logged in.

Save

After logging in, you can save a bracket for later viewing using the Keep button. You can view your saved tournaments using the My Brackets page.

Bookmarks

Another option is to simply bookmark individual tournaments using your browser.
