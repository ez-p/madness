Introduction

This program will randomnly select a NCAA bracket by simulating the tournament using a custom engine to calculate the winner. The engine has been tweaked according to both current year stats and historical tournament stats.

You can manually select the winner and/or runner up using the options page. The program will run simulations until a result that matches your selections is obtained.

Engine

Using the default engine "Seed Odds" the team with the higher seed has the better chance of winning a particular game. The higher seed's chance of winning increases with the seed spread. The number of upsets generated by this engine trends toward the 20 year average of 18 when running hundreds of thousands of simulations.

Alternative Engine

For demonstration purposes only, the winner is determined by a coin flip. This engine is not recommended. Alternative engines can be selected on the the options page page.

Madness

The algorithm can be adjusted using the 'madness' parameter on the the options page page. By default the madness variable is set to one. As the madness variable increases so does the tournament entropy. When adjusted up, lower seed teams have an increasing chance of winning against a higher seed.

The default madness level of one creates the average amount of madness seen over the last 25 years. Increasing the madness past 2 can create some truly crazy brackets! For example a madness level 3 starts to show a large number of 4,5,6 seeds in the final four, and you can even see some 7+ seeds in the final four.

Power

Internal to the system a 'power' level is associated with each team. This is adjusted each year according to the specific parameters of the teams playing in the tournament. For especially strong teams the power level is adjusted up. For high ranked weak teams, the power rating is adjusted down.

The default power is zero. Examples of adjusting the power level up is the undefeated 2015 Kentucky Wildcats which warrants a power level of five!

An example of adjusting the power level down is number one ranked 2016 Oregon who although earned a number one ranking was particularly weak when compared to the other number one teams.

The power level is tweaked for a particular year to resemble the rankings at www.fivethirtyeight.com.
